# Transport Service Reliability Analytics Dashboard
## Project Overview

This project automates the collection, transformation, and analysis of transport network status data using the Transport for London (TfL) API.

The solution provides operational insights into service disruptions, line performance, disruption severity, and daily service status through Python-based data pipelines and Power BI dashboards.
## Business Problem

Transport operators require visibility into service disruptions, reliability trends, and operational performance across multiple transport modes.

Raw API data is difficult to analyze directly. This project transforms live transport status data into structured datasets and actionable dashboards to support operational monitoring and decision-making.
